Java Interface 可用接口列表测试

  • 弹出提示框
    代码：
    HostApp.showToast('我是消息提示框');

  • 显示一个带有确认/取消按钮的消息提示框
    代码：
    HostApp.showAlertDialog('消息提示', '这是一个带有确认/取消按钮的消息提示对话框', function(){HostApp.showToast('响应了确认事件');});

  • 显示一个只带有确认按钮的消息提示框
    代码：
    HostApp.showAlertDialogForSingleBut('消息提示', '这是一个带有确认/取消按钮的消息提示对话框', function(){HostApp.showToast('响应了确认事件');});

  • 显示一个进度提示对话框
    代码：
    HostApp.showProgressDialog('正在拼命为您搬运数据中..', false);

  • 销毁一个正在显示中的进度提示对话框
    代码：
    HostApp.dismissProgressDialog();

  • 显示一个只带有多个选项的消息提示框
    代码：
    HostApp.showAlertDialogForItems('附件选择', '相机拍照,相册选择,文件浏览', function getItem(item, position){});

  • 打开新页面
    代码：
    HostApp.openNewpage('com.zzstxx.dc.parent.actions.NewShowActivity', '这是一个新页面', 'http://www.baidu.com', true); 
                /**
            * 打开一个新页面，同时允许设定是否要关闭当前页面
            *
            * @param pageClsName        要被打开的新页面类全名
            * @param title              新页面的标题
            * @param url                在页面中要打开的链接地址
            * @param isCloseCurrentPage 是否要关闭当前页面
            */

  • 打开不用传类名的新页面
    代码：
    HostApp.openNotclsNewpage('这是一个不需要传入类名就可以打开的新页面', 'http://www.baidu.com');

  • 打开可以播放视频的新页面
    代码：
    HostApp.openVideoPlayerPage('这是一个视频播放页', 'http://iclass.zzedu.net.cn/event.jsp?do=zyShow&id=5096');

  • 打开一个专用于带有附件上传的表单新页面
    代码：
    HostApp.openFormPage('这是一个带有附件上传的表单页面', 'http://iclass.zzedu.net.cn/event.jsp?do=zyShow&id=5096');

  • 打开带有自定义参数的新页面
    代码：
    HostApp.openNewpage('com.zzstxx.dc.parent.actions.NewShowActivity', '这是一个带有自定义参数的新页面', 'http://www.baidu.com', '这是自定义参数', false); 
                /**
            * 打开一个新页面，同时允许设定是否要关闭当前页面
            *
            * @param pageClsName        要被打开的新页面类全名
            * @param title              新页面的标题
            * @param url                在页面中要打开的链接地址
            * @param params             自定义的参数
            * @param isCloseCurrentPage 是否要关闭当前页面
            */

  • 刷新当前页面
    代码：
    HostApp.refreshCurrentPage('http://iclass.zzedu.net.cn/event.jsp?do=zyShow&id=5096');

  • 回退至上一级页面并关闭当前页
    代码：
    HostApp.goBackBeforePage();

  • 关闭当前页并回到首页
    代码：
    HostApp.goHomePage("com.zzstxx.hybrid.actions.MainActivity");

  • 清除当前页面Cookies
    代码：
    HostApp.clearCookies();

  • 获取当前设备的IMEI号码
    代码：
    HostApp.getDeviceIMEI();

  • 设置当前已登录用户的信息
    代码：
    HostApp.setUserInfo('登录信息，JSON字符串');

  • 获取当前登录用户的信息
    代码：
    var userinfo = HostApp.getCurrentUserInfo();
